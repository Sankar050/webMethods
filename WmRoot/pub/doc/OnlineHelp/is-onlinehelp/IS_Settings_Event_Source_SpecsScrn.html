Use the Event specifications page to view all of the event specifications for the Integration Server or filter the event specifications by streaming connection alias. An event specification defines the structure, topic, and encoding/decoding method for an event.
Click Create event specification to create an event specification.
Click Refresh Page to refresh the list of event specifications.
Event Specifications for Connection Alias
Name
Name for the event specification.
Connection Alias
Streaming connection alias for the streaming server to which events that are instances of this specification will be sent.
Topic
Topic on the streaming provider in which events are stored.
Key
Data type for the event key. The selected type indicates how Integration Server encodes the key object in a published event to bytes and how Integration Server decodes the bytes in the key object of an event received by an event trigger.
Value
Data type for the event value. The selected type indicates how Integration Server encodes the value object in a published event to bytes and how Integration Server decodes the bytes in the key object of an event received by an event trigger.
Delete
Click to delete the event specification.
