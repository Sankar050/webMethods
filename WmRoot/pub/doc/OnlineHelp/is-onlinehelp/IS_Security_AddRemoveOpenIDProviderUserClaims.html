Add and Remove OpenID Provider User Claims
Use the Add and Remove OpenID Provider User Claims page to assign, edit, or remove claims associated with an OpenID Provider user.
Under OpenID Provider Users, click Edit in the row for the user for which you want to edit, assign, or remove claims.
Click Return to Add and Remove OpenID Provider Users to return to the list of the OpenID Provider users that are currently defined on your server or to create or remove an OpenID Provider user.
Click Add ID Token Claim to add an ID Token claim for the user.
Click Add UserInfo Token Claim to add a UserInfo Token claim for the user.
