Connection Alias Settings
Displays the streaming connection aliases that are configured for the Integration Server
Name
Name of the streaming connection alias.
Type
The event streaming provider.
Description
Description of the streaming connection alias.
Test Connection
Click to test the connection to the streaming provider.
Enabled
Indicates whether or not the streaming connection alias is enabled. When a streaming connection alias is enabled, Integration Server can use it to send and receive messages from the streaming provider server on behalf of publishing services and event triggers, respectively. A disabled streaming connection alias cannot be used to send messages to or receive messages. To disable a streaming connection alias, click the Yes link in this column. To enable a streaming connection alias, click the No link in this column.
Status
The status of the streaming connection which can be “Running”, “Starting”, or “Not Running”.
Delete
Click to delete the streaming connection alias.
